Explaination

Each button has a function attached to them creating an interaction on the page.

After clicking create new element it takes the tagname (h3, div, a, etc) along with the class, id and innerHTML and creates a object directly on the page

parent id allows you to put an object into a div with a specific id already attached to it.

Remove last Element button does exactly what it says it does and removes the last element

Reset sets all the boxes with a for loop to be blank

the X on the right when clicked will hide the panel and add a button that shows the panel again and hides the show button.


Example

Enter div for the tagName, something like "border border-danger" in the className to use bootstrap borders, then enter example for id and text like hello world just to have something in the div. Then click create New Element

Now enter h3 for the tagName, Enter text such as "This is an h3 in a div, then enter example in the parent id box. Finally create New Element and you will see the h3 inside the div
